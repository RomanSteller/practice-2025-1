О проекте

Проект выполнен в рамках дисциплины «Проектная деятельность». Основной целью было создание современной системы взаимодействия с клиентами с использованием веб-технологий и Telegram-бота.

Описание проекта

Проект включает: разработку сайта и Telegram-бота для компании по металлообработке.


Основная часть — сайт на HTML. Дополнительно разработан Telegram-бот на Python (aiogram + Telegram Bot API).
