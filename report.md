## Введение

В рамках дисциплины «Проектная деятельность» нами был выполнен проект по разработке информационного веб-сайта и Телеграм-бота по теме **«Металлообработка»**.

Целью проекта стало создание удобных цифровых инструментов, которые наглядно представляют информацию по изучаемой теме и демонстрируют практические навыки разработки.

---

## 1. Общая информация о проекте

### 1.1 Название проекта
Разработка информационного веб-сайта и Телеграм-бота по теме проектной деятельности «Металлообработка».

### 1.2 Цели и задачи проекта

**Цели:**
- Создать современные цифровые продукты (сайт и Телеграм-бот) по теме металлообработки;
- Освоить HTML, CSS, JavaScript и разработку Телеграм-ботов;
- Получить опыт командной разработки.

**Задачи:**
- Разработать адаптивный статический сайт;
- Создать функционального Телеграм-бота;
- Реализовать удобную навигацию и оформление;
- Провести тестирование и документирование проекта.

---

## 2. Общая характеристика проектной деятельности

В рамках проектной деятельности мы изучали тему **«Металлообработка»**. Были рассмотрены основные виды обработки металлов:

- Токарная и фрезерная обработка;
- Лазерная резка металла;
- Сварочные работы;
- Гибка листового металла;
- Шлифовка и полировка изделий.

Для демонстрации полученных знаний и практического применения навыков программирования было решено разработать информационный веб-сайт и Телеграм-бота.

---

## 3. Описание задания по проектной практике

Согласно заданию требовалось выполнить базовую часть практики, включающую:

- Настройку Git-репозитория;
- Оформление материалов в формате Markdown;
- Создание статического веб-сайта (HTML + CSS + JavaScript);
- Разработку Telegram-бота;
- Подготовку отчёта.

Сайт должен содержать разделы: Главная, О проекте, Участники, Журнал работы, Ресурсы.
