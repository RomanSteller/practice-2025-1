## Задание

Разработка информационного веб-сайта и Телеграм-бота по теме проектной деятельности **«Металлообработка»**.

---

## О проекте

В рамках проектной практики нами разработан **статический веб-сайт** и **Телеграм-бот** по теме металлообработки.

Сайт выполнен в современном индустриальном стиле, полностью адаптивен.  
Телеграм-бот позволяет удобно получать информацию об услугах, ценах и оставлять заявки.

**Ссылка на Телеграм-бота:** [t.me/Meltallobot](https://t.me/Meltallobot)

---

## Используемые технологии

- **Frontend:** HTML5, CSS3 (Flexbox, Grid, анимации), JavaScript
- **Backend (бот):** Python 3, aiogram
- **Контроль версий:** Git + GitHub
